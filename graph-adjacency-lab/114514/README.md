# 景区路径规划系统 - 邻接矩阵实现

## 步骤一：解题思路

### 1.1 数据结构设计

```
┌─────────────────────────────────────────────────────────────────┐
│                        Graph (图)                               │
├─────────────────────────────────────────────────────────────────┤
│  spots[]     : ScenicSpot*   // 景点数组                        │
│  adj_matrix  : int**         // 邻接矩阵（存储路径距离）         │

│  desc  : char[100]    // 景点描述                               │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 邻接矩阵说明

邻接矩阵是一个 n×n 的二维数组，其中 n 为景点数量：
- `adj_matrix[i][j] = distance` 表示景点 i 到景点 j 的距离
- `adj_matrix[i][j] = INF` 表示景点 i 到景点 j 无直接路径
- `adj_matrix[i][i] = 0` 表示景点到自身的距离为0

### 1.4 核心函数伪代码

#### graph_create(capacity)
```
1. 分配Graph结构体内存
2. 初始化size=0, capacity=传入值
3. 分配spots数组内存 (capacity * sizeof(ScenicSpot))
4. 分配邻接矩阵内存 (capacity * sizeof(int*))
5. 循环i从0到capacity-1:
    分配adj_matrix[i]数组内存 (capacity * sizeof(int))
    循环j从0到capacity-1:
        if i == j: adj_matrix[i][j] = 0
        else: adj_matrix[i][j] = INF
6. 返回graph指针
```

#### graph_add_spot(graph, id, name, desc)
```
1. if graph->size >= graph->capacity: return -1 (已满)
2. if find_spot_index(graph, id) != -1: return -2 (重复)
3. 复制id到graph->spots[size].id
4. 复制name到graph->spots[size].name
5. 复制desc到graph->spots[size].desc
6. graph->size++
7. return 0 (成功)
```

#### graph_add_path(graph, from_id, to_id, distance)
```
1. from_idx = find_spot_index(graph, from_id)
2. to_idx = find_spot_index(graph, to_id)
3. if from_idx == -1 || to_idx == -1: return -1 (景点不存在)
4. graph->adj_matrix[from_idx][to_idx] = distance
5. graph->adj_matrix[to_idx][from_idx] = distance  // 无向图
6. return 0 (成功)
```

#### graph_display(graph)
```
1. 打印表头行（所有景点ID）
2. 循环i从0到graph->size-1:
    打印行号（景点ID）
    循环j从0到graph->size-1:
        if adj_matrix[i][j] == INF: 打印 "-"
        else: 打印 adj_matrix[i][j]
```

#### graph_display_spots(graph)
```
1. 打印"景点链表"标题
2. 循环i从0到graph->size-1:
    打印景点序号、ID、名称、描述
```
